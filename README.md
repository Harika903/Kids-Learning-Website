# Kids-Learning-Website
An interactive kids learning website developed using HTML, CSS, and JavaScript, featuring quizzes, games, stories, rhymes, and drawing activities. The platform enhances children's learning through audio-visual content and engaging educational exercises.

## Features
- A-Z learning with audio
- Interactive quiz and games
- Drawing & coloring page
- Rhymes and story section
- Rewards system

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Structure
- HTML pages for each section
- Images, sounds, and videos for learning
- Interactive UI for kids engagement

## Outcome
Designed an engaging platform to make learning fun and interactive for children.
